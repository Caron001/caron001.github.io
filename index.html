My Projects

Web Cinema

This project is a cinema web platform developed using Laravel, with the integration of the TMDb (The Movie Database) API to provide comprehensive and up-to-date movie information. User data is stored and managed using an SQL database, ensuring security and reliability in data storage. This cinema web platform features movie search, movie details, user management, reviews and ratings, as well as an admin panel. I developed this project to participate in a web development competition, aiming to create an intuitive and engaging application for users while providing accurate and comprehensive movie information. In addition to the competition, this project also serves as a learning tool and an opportunity to apply skills in modern web development, offering real-world experience in building complex and fully functional web applications.

Online Shop

KicksConnect is an online shoe shop website that I created with three other teammates as our final project for the third semester. This website is developed using Laravel for the backend, Bootstrap for the frontend, and SQL for the database, ensuring a responsive, secure, and reliable application. KicksConnect offers various features such as an organized product catalog, a shopping cart system, a wishlist for saving favorite products, and user management, providing a comfortable and intuitive shopping experience for users. Additionally, there is an admin panel that allows admins to add new products, edit product information, and view and manage customer orders.
